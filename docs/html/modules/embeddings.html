classmethod load_vectors(path: str, embedding_type: str, fine_tuned: bool = False) → Embeddings[source]#

Instantiate Embeddings with pre-trained FastText or Word2Vec models.

Parameters:

  • path (str) – Location of the embeddings to load.

  • embedding_type (str) – The type of embedding vectors. One of {fasttext, word2vec}.

  • fine_tuned (bool, optional) – Whether the embedding vectors have been fine tuned. Matters if embedding_type=’fasttext’ since self-trained or fine tuned FastText models need to be read differently than the official ones from [https://fasttext.cc/].

Example

>>> embeds = Embeddings.load_vectors('path/to/embeddings')  
...
Raises:

ValueError – If an invalid embedding_type is entered.

Returns:

Embeddings object.
